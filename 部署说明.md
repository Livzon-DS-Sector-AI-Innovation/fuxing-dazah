前端
pnpm build
docker build --platform linux/amd64 -t dazah-frontend:v[版本号] .
docker save -o dazah-frontend.tar dazah-frontend:v[版本号]
服务器
查看dazah-frontend容器id
docker ps
移除旧版本
docker stop id
docker rm id
docker rmi 旧版本镜像
docker load -i dazah-frontend.tar
docker run -d -p 3000:3000 -e API_BASE_URL=http://192.168.3.100:8000 --name dazah-frontend dazah-frontend:v[版本号]

后端
服务端更新代码
如果有变更项目依赖或数据库，执行
docker compose --profile app down
docker compose --profile app up -d --build
只是app代码更新，执行
docker compose restart app
重载环境变量，执行
docker compose --profile app up -d --force-recreate
